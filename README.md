# Batman Logo Medal

## Description

This project is a simple **Batman Logo Medal** designed in **Onshape** and exported as an **STL** file. It is intended for 3D printing and can be used as a decorative item or display piece.

## Software

- Onshape

## Onshape link

## File Format

- STL (`.stl`)

## Recommended Print Settings

- **Material:** PLA
- **Layer Height:** 0.2 mm
- **Infill:** 20%
- **Supports:** Not required

## Purpose

- Practice 3D modeling and printing
- Decorative item
- Batman-themed display piece
