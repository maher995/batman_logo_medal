#Batman Logo Medal – STL Design
Overview

This project is a 3D printable Batman Logo Medal designed in Onshape and exported as an STL file. The model features the iconic Batman emblem in a simple, flat medal-style design, making it suitable for decorative purposes, keychains, display pieces, or beginner 3D printing projects.

Features
Classic Batman logo silhouette
Lightweight, flat medal design
Easy to 3D print without supports (depending on print orientation)
Suitable for FDM and resin 3D printers
Exported in STL format for compatibility with most slicing software
Software Used
CAD Software: Onshape
Export Format: STL
Recommended Print Settings
Layer Height: 0.2 mm
Infill: 15–20%
Wall Count: 2–3 perimeters
Supports: Not required for most printers
Print Material: PLA (recommended), PETG, or ABS
Applications
Decorative medal
Keychain (if a hole is included or added)
Desk ornament
Gift or collectible
Educational CAD and 3D printing project
File Information
File Type: .stl
Model: Batman Logo Medal
Designed In: Onshape
